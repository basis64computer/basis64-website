Anggota_Kelompok
Deskripsi:
1. Membuka website melalui browser
2. Menampilkan halaman awal (Dashboard)
3. Jika mengklik produk
 Menampilkan list perangkat elektronik sesuai kategori yang dipilih
4. Jika mengklik Jasa
 Menampilkan jasa sesuai kategori jasa yang dipilih
5. Jika mengklik Software
 Menampilkan list software sesuai kategori software yang dipilih
6. Jika mengklik Feedback
 Menampilkan halaman feedback
 Pengguna memasukkan nama dan pesan
 Jika mengirim feedback
  Script feedback.js akan memproses input berisi nama dan pesan dari pengguna dan mengirimnya ke API bot telegram
  Jika berhasil
   Menampilkan "Masukkan Anda berhasil dikirim."
  Jika gagal
   Menampilkan "Masukkan Anda tidak dapat dikirim."
7. Jika mengklik Anggota_Kelompok
 Menampilkan list anggota kelompok
Pseudocode
START
 kategori, dashboard, produk:ARRAY, jasa:ARRAY, software:ARRAY, feedback:ARRAY, anggota_kelompok, feedback_js, bot_telegram, nama, pesan

IF (mengklik produk)
 MENAMPILKAN produk[kategori]
IF (mengklik jasa)
 MENAMPILKAN jasa[kategori]
IF (mengklik software)
 MENAMPILKAN software[kategori]
IF (mengklik feedback)
 MENAMPILKAN feedback
 INPUT nama
 INPUT pesan
 IF (mengirim feedback)
  MENJALANKAN feedback_js
  feedback_js memproses nama, pesan
  feedback_js mengirim nama, pesan ke bot_telegram
  IF (BERHASIL)
   MENAMPILKAN "Masukkan Anda berhasil dikirim."
  ELSE
   MENAMPILKAN "Masukkan Anda tidak dapat dikirim."
IF (mengklik anggota_kelompok)
 MENAMPILKAN anggota_kelompok
END
